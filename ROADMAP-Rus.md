## Фаза 1 — Базовые утилиты

### ✅ Серия (Огонёк)
> Выпущено в [v1.3.0](https://github.com/vvertax/DailyTimeTracker/releases/tag/v1.3.0)

- **Иконка огня** в шапке popup-а с цветным счётчиком серии.
- **Серия видна** начиная с 2+ последовательных дней.
- **Счётчик серии** рядом с иконкой (например, `🔥 5`).
- **Пульсирующее свечение** для серий 3+ дней.
- **Серия сбрасывается**, если пропущен полный день (0 секунд).
- Минимальный порог для учёта дня: **5 мин** прослушивания.
- Данные серии хранятся в LocalStorage.

### ✅ Экспорт в CSV / JSON
> Выпущено в [v1.3.0](https://github.com/vvertax/DailyTimeTracker/releases/tag/v1.3.0)

Скачивание данных одним кликом из панели настроек.

- **CSV** — дата, секунды, форматированная длительность — готово для Excel / Google Sheets.
- **JSON** — полный дамп включая интервалы по дням — для разработчиков или бэкапа.
- Область экспорта: вся сохранённая история + сегодня.

### ✅ Очистка / Сброс данных
> Выпущено в [v1.3.0](https://github.com/vvertax/DailyTimeTracker/releases/tag/v1.3.0)

Деструктивное действие за диалогом подтверждения в настройках.

- **Сбросить сегодня** — очистить текущий день, сохранить историю.
- **Очистить историю** — удалить архивные дни, сохранить сегодня.
- **Полный сброс** — удалить все сохранённые данные.

## Фаза 2 — Аналитика и инсайты

### 📊 Недельная сводка
Новый переключатель в popup: **Сегодня** / **Неделя**.

- Общее время прослушивания за последние 7 дней.
- Среднее в день.
- Самый активный день недели.
- Мини-график или спарклайн с totals по дням.

### 📈 Сравнение с прошлой неделей
Быстрый блок сравнения: **эта неделя** vs **прошлая неделя**.

- Разница по времени (часы/минуты).
- Процент изменения с индикатором роста/падения.
- Короткая подсказка вроде «Вы опережаете прошлую неделю».

###  Разбивка по дням недели
Агрегированная статистика по дням недели за весь период хранения.

- Среднее по дням (Пн–Вс).
- Сравнение будней и выходных.
- Подсветка «самого активного дня недели».

### 🧭 Индикатор темпа прослушивания
Показывает, идёт ли текущий месяц выше или ниже вашего обычного темпа.

- Текущее среднее в день за месяц.
- Базовое среднее по сохранённой истории.
- Метка «в темпе» для быстрого чтения в компактном UI.

## Фаза 3 — Продвинутые функции

### 🛑 Тепловая карта по часам (Отменено)
24 колонки (0:00–23:00), показывающие пиковые часы прослушивания.

- Источник данных: существующие интервалы.
- Период: последние 7 дней или весь период хранения (переключатель).
- Компактный SVG-график внутри popup.

### ✅ Импорт из JSON
> Выпущено в [v1.6.0](https://github.com/vvertax/DailyTimeTracker/releases/tag/v1.6.0)

Восстановление или слияние ранее экспортированных данных из панели Settings.

- Выбор файла в настройках.
- Режимы импорта: **Merge** (безопасный) или **Replace** (полная замена).
- Стратегия слияния: объединение интервалов, выбор большего total для пересекающихся дней.
- Валидация: отклонение некорректных или подозрительно больших файлов.
- Шаг предпросмотра перед применением (затронутые дни, конфликты, totals).

### 🎛️ Компактный / Полный режим popup
Два варианта интерфейса — чтобы UI не разбухал с ростом функций.

**Компактный (по умолчанию):**
- Total за сегодня + прогресс огонька.
- Последние 1–2 сессии.
- Краткая история (3 дня).

**Полный (по переключателю):**
- Все сессии.
- Недельная сводка + серии.
- Полная история.
- Доступ к настройкам.

## Опциональные / Нишевые функции

### 🌍 Граница дня с учётом часового пояса
Ручная настройка часового пояса для тех, кто путешествует или живёт по нестандартной границе дня.

- Смена дня и логика streak привязаны к выбранному часовому поясу.
- Пресеты + кастомный UTC-сдвиг.
- Безопасная миграция, чтобы не потерять текущую историю.

## Вне текущего скоупа

| Идея | Почему отложено |
|---|---|
| Облачная синхронизация | Требует инфраструктуру авторизации; LocalStorage достаточен для одного устройства. |
| Аналитика по трекам | Spicetify Player API отдаёт play/pause, но не надёжные метаданные трека для архива. |
| Синхронизация между окнами | LocalStorage не транслирует события между экземплярами Spotify надёжно. |
| Тяжёлые библиотеки графиков | Размер бандла и runtime-нагрузка не оправдывают ценность для лёгкого расширения. |
