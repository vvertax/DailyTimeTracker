# Daily Time Tracker для Spicetify

Этот скрипт отслеживает, сколько времени вы проводите в Spotify каждый день, и показывает таймер в верхней панели.

Этот код был написан с помощью ChatGPT.

## Возможности

- Таймер в верхней панели Spotify
- Формат: `ЧЧ:ММ:СС` или `ДД:ЧЧ:ММ:СС`
- Автопауза после 30 секунд без воспроизведения
- Ежедневный сброс в полночь с сохранением истории
- Всплывающее окно с историей по дням при наведении
- Клик для закрепления или открепления окна
- Переключатель языка: `RU / ENG`
- Локальное хранение истории через `Spicetify.LocalStorage`

## Текущий интерфейс

- Таймер в верхней панели
- Всплывающее окно с историей по дням в формате `ГГГГ-ММ-ДД - ЧЧ:ММ:СС`
- Строка с сегодняшней датой подсвечивается
- По клику на таймер или окно оно закрепляется

## Файлы

- [`DailyTimeTracker.js`](./DailyTimeTracker.js): основное расширение Spicetify

## Установка

### Windows

1. Поместите `DailyTimeTracker.js` в:
   `C:\Users\addiction\AppData\Roaming\spicetify\Extensions`
2. Подключите расширение:

```powershell
spicetify config extensions DailyTimeTracker.js
spicetify apply
```

## Поведение

- Если музыка играет, время увеличивается
- Если воспроизведение остановлено меньше чем на 30 секунд, счёт продолжается
- Если воспроизведение остановлено на 30 секунд или дольше, счёт ставится на паузу
- При закрытии Spotify текущий прогресс сохраняется
- При наступлении нового дня предыдущий день автоматически переносится в архив

## Хранение данных

Скрипт использует следующие ключи `Spicetify.LocalStorage`:

- `dtt_today_v3`: данные текущего дня
- `dtt_history_v2`: архив истории по дням
- `dtt_language_v1`: выбранный язык интерфейса

## Примечания

- Код предоставляется как есть
- Изменения в DOM Spotify или Spicetify могут потребовать будущих обновлений
- Загрузка внешнего шрифта может работать по-разному в зависимости от окружения
