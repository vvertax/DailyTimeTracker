# Daily Time Tracker для Spicetify

Расширение считает, сколько времени вы слушаете Spotify каждый день, и показывает таймер в верхней панели.

> [!NOTE]
> Этот код был написан с помощью ChatGPT.

![Превью](./preview.png)

## Возможности

- **Таймер в верхней панели:** Счетчик в формате `ЧЧ:ММ:СС` или `ДД:ЧЧ:ММ:СС`.
- **Более точный трекинг сессий:** Сессии опираются на реально прослушанное время, а не на сырое время паузы, и завершаются аккуратнее, когда воспроизведение действительно останавливается.
- **Настраиваемый порог паузы:** В Settings можно выбрать `15 / 30 / 60 / 120 сек`.
- **Подробная история:** Popup с сессиями за сегодня и архивом по дням.
- **Top Tracks Today:** Необязательный блок под общим временем за день с самыми часто проигрываемыми треками за текущий день. Прослушивание засчитывается только после минимум `20` секунд реального воспроизведения.
- **Недельная сводка + Yesterday vs Today:** Кнопка `Неделя` показывает последние 7 дней, среднее, лучший день, мини-график и быстрый сравнительный режим с вчерашним днем.
- **Режимы Compact / Full:** Compact делает popup короче, а Full показывает все сессии, всю историю и недельный переключатель.
- **Режим производительности:** Можно выбрать `Default` или `Lightweight`, чтобы уменьшить нагрузку popup и визуальных эффектов на слабых системах.
- **Daily Goal:** В Settings можно задать дневную цель по времени прослушивания и видеть прогресс прямо в popup.
- **Хранение истории + Forever:** Историю можно хранить от `1` до `6` месяцев или отключить автоочистку через `Forever`.
- **Огонек серии:** Счетчик серии появляется от `2+` подряд идущих дней, которые достигли минимального порога.
- **Streak Shields + Keep Streak:** До `4` пропущенных дней в календарном месяце могут автоматически защитить серию, а опция `Keep streak` сохраняет текущую серию без роста.
- **Длинная прогрессия серии:** Дополнительные цветовые уровни для длинных streak.
- **Каналы Release / Test / Dev:** Канал можно переключать из Settings. Release стабильнее, а Test и Dev подходят для более ранних изменений.
- **Ручная проверка обновлений:** Обновления можно проверить из Settings, а текст версии работает как быстрый триггер проверки.
- **Экспорт в CSV / JSON:** Выгрузка всей сохраненной истории и текущего дня из Settings.
- **Импорт из JSON:** Восстановление или объединение предыдущего JSON-экспорта с предпросмотром и валидацией.
- **Очистка и сброс:** Можно сбросить сегодня, очистить историю или сделать полный сброс с подтверждением.
- **Переключение языка:** `RU` и `EN` прямо внутри popup.
- **Пользовательские бейджи:** Необязательный серверный бейдж рядом с заголовком popup.
- **Удаленные runtime-обновления:** Marketplace-лоадер подтягивает основной код с `https://vvertax.site/dtt/ext/main.mjs` и проверяет обновления через `https://vvertax.site/dtt/ext/version.json`.

## Текущий интерфейс

- **Виджет в верхней панели:** Показывает текущее время за сегодня.
- **Шапка popup:** Заголовок, необязательный бейдж, огонек серии, кнопка `Неделя`, зависящая от текущего режима, и кнопка Settings.
- **Раздел Today:** Общее время за день, необязательный прогресс цели, необязательный блок top tracks, сессии за сегодня и сохраненная история.
- **Раздел Week:** Недельная сводка со средним, лучшим днем, мини-графиком и подрежимом `Yesterday vs Today`.
- **Панель Settings:** Язык, канал, режим popup, режим производительности, переключатель top tracks, количество top tracks, порог паузы, щиты серии, `Keep streak`, daily goal, срок хранения истории, длинная прогрессия серии, видимость бейджика, экспорт, импорт, действия сброса, ручная проверка обновлений и текст версии.

## Установка

### Через Spicetify Marketplace

1. Откройте Spicetify Marketplace.
2. Найдите `Daily Time Tracker` во вкладке **Extensions**.
3. Нажмите **Install**.

### Ручная установка

1. Скачайте `DailyTimeTracker.js` [здесь](marketplace/DailyTimeTracker.js).
2. Поместите файл в папку расширений Spicetify:
   `%AppData%\spicetify\Extensions` на Windows или `~/.config/spicetify/Extensions` на Linux/macOS.
3. Выполните:

```powershell
spicetify config extensions DailyTimeTracker.js
spicetify apply
```

## Примечания

- Очистка истории затрагивает видимый сохраненный архив, но серия все равно считается по сырым данным истории.
- Данные top tracks привязаны к текущему дню и сбрасываются при начале нового дня.
- После переключения канала требуется перезагрузка Spotify.
- Архитектура с удаленным runtime позволяет выпускать обновления быстрее, без ожидания Marketplace review.
- Расширение зависит от DOM-структуры Spotify и Spicetify, поэтому изменения интерфейса Spotify могут потребовать фиксов.
